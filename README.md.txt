# 💥 DDoS Detection using Machine Learning

This project focuses on detecting Distributed Denial of Service (DDoS) attacks using machine learning techniques. It applies models like SVM, Random Forest, and K-Means for accurate detection of malicious traffic.

## 🔍 Features

- Real-time DDoS attack detection
- Model training using Python and Scikit-learn
- Deployed using Django & Azure App Service
- CI/CD pipeline using GitHub Actions
- Secure integration with Azure Key Vault

## 📂 Tech Stack

- **Python** | **Scikit-learn** | **Pandas** | **NumPy** | **Matplotlib**
- **Django** for web interface
- **Azure App Service** for deployment
- **GitHub Actions** for CI/CD

## 🧠 ML Algorithms

- K-Means Clustering
- Support Vector Machine (SVM)
- Random Forest Classifier
